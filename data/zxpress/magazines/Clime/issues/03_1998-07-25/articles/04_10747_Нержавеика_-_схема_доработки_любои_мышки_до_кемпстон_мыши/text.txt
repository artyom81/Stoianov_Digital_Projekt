  Вот  решил черкануть пару, или больше,
строчек в первую сетевую газетку.

  Пишу  вот по какому поводу - уже в ко-
торый  раз  мне задают вопрос: "Чего там
всунуть внутрь крыса?" Отвечаю: "Впихни-
те  схему, которая нарисована ниже!" На-
деюсь, что те, которым это нужно, доста-
нут  этот  номер  газеты и перерисуют ее
(схему)  всем  тем, которым это не очень
надобно! ;-()

  А вот и сама схема:

  +5 вольт
     ^                    К1533ТЛ2
     |                     ______
     |                    |      |
     | а это один         |      |
  | / из транзис-    _____|      |_____
 _|/<-торов опто-   /     |      | выход
  |  вой пары!  | /      |      |
  | V            |/       |______|
     +-----------|  КТ315
     |           |
     |           | 
    ---             V
    | |             |
    | | 5,1К        |
    | |             |
    | |             |
    ---             |
     |              |
     +--------------+
                    |
                  --+--

  Оптопара в своем составе имеет два фо-
тотранзистора.  Средний  вывод  - общий,
его  надо  подключать к +5в. Два крайних
подключаются  к  общей  точке соединения
резистора и базы транзистора. Естествен-
но, что не оба сразу. Для каждого фотот-
ранзистора  нужна  своя  такая схема - в
итоге  у вас будет четыре резистора, че-
тыре  транзистора  и  одна микросхема. С
выходов микросхемы берем сигналы направ-
лений.

  Светодиоды (это которые стоят напротив
оптопар) должны быть включены последова-
тельно,  вместе с резистором 150-300 Ом,
и  все это подключается к общему проводу
питания и к +5 вольт.

  Если после подключения у вас направле-
ние(я)  движения  стрелки  (или  чего-то
еще)  противоположны  движениям мыши, то
поменяйте местами выходы Xa и Xb, или Ya
и Yb, или и те, и другие.

  О  замене  деталей. Вместо резистора с
таким  сопротивлением уже ничего не пос-
тавишь другого, т.к. его номинал был по-
добран  мной лично методом научного тыка
с  использованием осциллографа. Транзис-
тор  любой  малогабаритный со структурой
n-p-n. Например КТ315, КТ3102. Ну если у
кого  - нибудь в мышке навалом места, то
он   может   попытаться  затолкать  туда
КТ819БМ,  или  КT903,  или КТ908, или не
помню что еще. Микросхема серии 1533 за-
меняется  на  555. Можно попробовать 155
или  561.  Правда  я сам такой замены не
производил.  Но 561я стояла в схеме, на-
печатанной  то ли в "Спектрофоне", то ли
в "Формате".

  И еще немного о плате интерфейса. Вхо-
ды для кнопок нужно подтянуть резистора-
ми  к  +5в.  Сопротивление от 1к до 30к.
Входы направлений подтянуть к общей шине
питания. Резисторы те же.

  Теперь  информация для тех, у кого нет
справочника по микросхеме.

   Входы - 1,3,5,9,11,13.
  Выходы - 2,4,6,8,10,12
    +5B  - 14
   общий - 7

  Думаю, что информация исчерпывающая, и
любой  сможет  доработать  любую мышь до
кемпстон мыши.
