            СООБЩЕHИЕ О Microsoft Windows BS Extra
           ----------------------------------------

     В ближайшее время начнется распространение нового продук-
та Microsoft Corporation - Microsoft Windows  BS  Extra.  Этот
продукт является  естественным развитием MS Windows версий 3.0
и 3.1, отличающимся более развитыми системными возможностями и
дружественным к работникам Microsoft  Corporation  интерфейсом
пользователя.
     Инсталляция продукта  требует  165 Кб  дискового   прост-
ранства, 32 Мб памяти, процессора i486 или лучше. В силу свое-
го высокого качества Microsoft Windows BS Extra весьма  требо-
вательна к конфигурации, и, если этот пакет по каким-либо при-
чинам не запускается на вашем  компьютере,  фирма  рекомендует
менять машины до тех пор, пока она, наконец, не запустится.
     Улучшен интерфейс пользователя,  так,  например,  в целях
экономии экранной площади,  значок "Ok", в который следует по-
падать мышью, уменьшен в два раза, идет работа над уменьшением
его в четыре и более раз.  Введено,  помимо двойного (double),
тройное (triple) и четверное (quadruple) нажатие клавиши мыши.
При этом следует иметь  в  виду.  что  нажатие  остальных,  не
используемых программой клавиш, может привести к непредсказуе-
мым последствиям, и поэтому рекомендуется выламывать их из мы-
ши перед инсталляцией. Hа экран выводятся теперь сообщения обо
ВСЕХ  неожиданностях,  встреченных  программой в памяти или на
диске,  для доходчивости их прочтение следует  подтвердить  не
менее чем  пять раз.  Увеличено разнообразие неописанных типов
файлов, еще более специализировано содержание файлов инициали-
зации.  Поскольку увеличен объем инструкций пользователя,  от-
носящийся к тому, как нажимать клавиши, описания файлов иници-
ализации в документации опущены.
     Microsoft Windows BS Extra требует SVGA card,  работает в
режиме только  256  цветов,  хотя неопытный пользователь может
этого и не заметить - все графические  приложения  по-прежнему
используют 16-ти цветовую палитру - это так трогательно!
     Улучшен и многозадачный режим: размер кванта времени, вы-
деляемый каждой задаче унифицирован, и составляет одну минуту.
     Введено много  новых  стандартов,  включая   альтернативу
стандарту ASCII, и новой грамматики английского языка. Все они
недокументированные,  но, утверждает Билл Гейтс, это и не нуж-
но, работники Microsoft Corporation легко обходятся без допол-
нительных приложений, а, значит, обойдутся и остальные пользо-
ватели. Следует приучать пользователей к использованию продук-
тов только Microsoft Corporation,  а программистов - к  работе
только в ее рамках. Ввод-вывод и счет - вот все, что может по-
надобиться программисту в среде Microsoft  Windows  BS  Extra!
Все остальное  уже  сделано  Microsoft  Corporation.  Вряд  ли
кто-либо захочет нечто,  чего  не  смогли  предусмотреть  даже
программисты Microsoft Corporation!
     Сюрприз предстоит также и  неанглоязычным  пользователям:
дополнительные знаки алфавита их языка они найдут в новом, бо-
лее удобном Microsoft Corporation месте.
     Список программ,  конфликтующих с  Microsoft  Windows  BS
Extra,  не может быть здесь приведен, так как занимает 392 Кб.
Однако можно с уверенностью сказать,  что Microsoft Windows BS
Extra  входит  в  конфликт  со  всеми  программами  фирм IBM и
Symantec (последнее было достигнуто после  долгого  труда),  а
также многими другими.  Единственной рекомендацией в этом слу-
чае является немедленное удаление с жесткого диска всего,  что
не относится к Microsoft Windows BS Extra.
     В этой версии разработчики полностью отказываются от сле-
дования принципам MSDOS,  признавая ее своей ошибкой и тупико-
вым направлением. Соответственно, так как, по заверениям прог-
раммистов Microsoft Corporation,  функция выполнения ДОСовских
приложений из среды Windows не пользуется  среди  них  большой
популярностью, в новом продукте она более  не  поддерживается.
"Что годится для нас - сгодится и для вас" - таков новый девиз
Microsoft Corporation.
     Если система  Windows  NT  была   провозглашена   заменой
UNIX'у, то  Microsoft  Windows BS Extra,  по мысли руководства
Microsoft Corporation предстоит заменить вообще  все  мыслимые
операционные стстемы,  так она хороша. Все же слухи о том, что
BS означает "Big Shit" ("Большое  дерьмо"),  не  соответствуют
действительности.
