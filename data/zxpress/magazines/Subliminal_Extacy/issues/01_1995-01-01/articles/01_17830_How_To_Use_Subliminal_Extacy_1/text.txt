As you can see this version of Subliminal Extacy has got a      
slightly different outlook to it.  This is because the old menu 
and text viewer was a bit crappy!!!  Also in this version there 
are a few extra keys and the other keys are easier to cope with.
So here are the keys for Subliminal Extacy #1!                  
                                                                
                            Menu Keys                           
                            ---------
                           Q = Up Menu                          
                          A = Down Menu                         
                      Enter = View Article                      
                         F = Change Font                        
                        M = Change Music                        
                        E = Exit (USR 0)                        
                                                                
                       Article Viewer Keys                      
                       -------------------
                         Q = Up Article                         
                        A = Down Article                        
                     Space = Return To Menu                     
                         F = First page                         
                          L = Last page                         
                                                                
I'm sure you will agree that these keys are easier to cope with 
and the extra keys makes this Issue of Subliminal Extacy        
easier to use.  User friendliness is what Speccy needs!
