Что еще нового:
В  меню Disk options появились: полоса
прокрутки,  возможность грузить модули от
Pro  Tracker  2.1,  Sound  Tracker, Sound
Tracker  Pro, возможность записи/загрузки
редактируемого паттерна.                 
                               
   В меню Edit sample реализованы команды
накапливания    и   уменьшения/увеличения
громкости  звучания, маска для огибающей,
рисование амплитуды стрелкой.            
                               
   В  меню  Edit  Ornament  и Edit Sample
возможно  проигрывание любого сэмпла (ор-
намента)  с любым орнаментом (сэмплом) на
всех нотах и октавах.                    
                               
   Счетчик нот (для создателей trackmo)  
                               
   Чтобы выключить/включить канал, нажми-
те на анализаторе Space.
Управление:

управление стрелкой: Q, Caps Shift, O, P,
Space.

Опции основного экрана:
TEMPO +
Уменьшает скорость проигрывания мелодии. 
Минимальная - 63.
TEMPO -
Увеличивает скорость проигрывания мело-  
дии.
Максимальная - 3.
POSITION +
Увеличивает номер текущей позиции мело-  
дии.                                     
Максимальное количество позиций - 256.
POSITION -
Уменьшает номер текущей позиции мелодии.
PATTERN +
Увеличивает номер PATTERNa в текущей по- 
зиции.                                   
Максимальное количество паттернов - 42.
PATTERN -
Уменьшает номер PATTERNa в текущей пози- 
ции.
LENGTH +
Увеличивает длинну мелодии - количество  
проигрываемых позиций
LENGTH -
Уменьшает длинну мелодии.
LOOP TO +
Увеличивает номер позиции для зациклива- 
ния мелодии.
LOOP TO -
Уменьшает номер позиции для зацикливания 
мелодии.
VOLUME +
Увеличивает общую громкость звука.       
    Максимальное значение - 15.
VOLUME -
Уменьшает общую громкость звука.
OKTAVE +
Увеличивает номер текущей октавы.        
        Количество октав - 8.
OKTAVE -
Уменьшает номер текущей октавы.
EDIT PATTERN +
Увеличивает номер рабочего PATTERNa.
EDIT PATTERN -
Уменьшает номер рабочего PATTERNa.
PATTERN LENGTH +
Увеличивает длинну текущего PATTERNa.    
       Максимальная длинна - 64.
PATTERN LENGTH -
Уменьшает длинну текущего PATTERNa.      
      Минимальная  длинна - 5.
WRITE DATA ON/OFF
Aвтoматичeскя установка параметров нот.
SCROLL STEP +
Увеличение шага скроллирования при уста- 
новке ноты.                              
Максимальное значение - 8.
SCROLL STEP -
Уменьшение шага скроллирования при уста- 
новке ноты.                              
Минимальное значение - 1.
AUTO EN SET ON/OFF
Автоматическая/не автоматическая установ-
ка младшего байта периода огибающей.
DIVISION
Младший байт периода огибающей для AUTO  
EN.
DATA
Образец параметров ноты для автoматичeс- 
кой установки.
DISK OPTIONS
Загрузка/выгрузка  мелодии,  инструмента,
орнамента, паттерна. Расширение соответс-
твенно:  m,  s,  o,  р.  Используя полосу
прокрутки,  можно  перемещаться по списку
файлов.  По  умолчанию  грузятся  мелодии
формата: PTЗ, PT2, ST, ST Pro. Для полно-
го аналога их звучания требуется дoрабoт-
ка.
CLEARS
Очистка мелодии, позиций, паттернов, инс-
tpymehtob, орнаментов.
EDIT MELODY
Редактирование паттернов.
EDIT PATTERN
Редактирование  рабочего паттерна.
INFO ABOUT MUSIC
Название мелодии.                        
Максимальная длинна - 32 символа.
INFO ABOUT AUTHOR
Информация об авторе. Максимальная длинна
- 32 символа.
EDIT SAMPLE
Редактирование   инструмента.  Количество
инструментов - 31. Используя клавиши от 2
до 7, можно увеличивать значения парамет-
ров,  а  от  W до U - уменьшать. True/Inv
Video  - перемещение по инструменту прыж-
ками.  Розовая полоска на левой колонке -
зацикливание  инструмента.  T - установка
Tone,  N-  Noise,  E- Envelope (маски для
огибающей).  Для проигрывания инструмента
-  нотные  клавиши. Шум отображен цифрами
от  0 до 31, тон - горизонтальными полос-
ками,  которые  можно  рисовать стрелкой.
Также  реализованы команды накапливания и
уменьшения/увеличения громкости звучания.
EDIT SAMPLE +
Увеличение номера редактируемого инстру- 
мента.
EDIT SAMPLE -
Уменьшение номера редактируемого инстру- 
мента.
TO SAMPLE +
Увеличение номера инструмента-приемника  
для копирования.
TO SAMPLE -
Уменьшение номера инструмента-приемника  
для копирования.
COPY SAMPLE
Копировать редактируемый инструмент.
EDIT ORNAMENT
Редактирование  орнамента. Количество ор-
hamehtob  - 15. Розовая полоса - зацикли-
вание.
EDIT ORNAMENT +
Увеличение номера редактируемого ophameh-
та.
EDIT ORNAMENT -
Уменьшение номера редактируемого орна-
мента.
TO ORNAMENT +
Увеличение номера орнамента-приемника для
копирования.
TO ORNAMENT -
Уменьшение номера орнамента-приемника для
копирования.
COPY ORNAMENT
Копировать редактируемый орнамент.
TRANSPOSITION
Изменение высоты нот паттерна в полуто-  
нах.
TRANSPOSITION +
Увеличение приращения высоты нот. Макси- 
мальнoe значение: +64.
TRANSPOSITION -
Уменьшение приращения высоты нот. Мини-

COPY
Копирование блока нот.
FROM PATTERN +
Увеличение номера паттерна, содержащего  
исходный блок нот.
FROM PATTERN -
Уменьшение номера паттерна, содержащего  
исходный блок нот.
TO PATTERN +

ется блок нот.

блок нот.
TO CHANNEL -
Уменьшение номера канала, куда копируется

Увеличение номера первой ноты в исходном

FROM BEGIN -
Уменьшение номера первой ноты в исходном

Увеличение номера последней ноты в исход-

END -
Уменьшение номера последней ноты в исход-
ном блоке нот.
EXCHANGE
Поменять блоки нот местами.
PLAY MELODY
Играть мелодию. Выход - Extend.
PLAY PATTERN
Играть текущий паттерн. Выход - Extend.

Откомпилировать мелодию. Младший байт ад-
реса  старта всегда равен нулю. Появились
ячейки  Data  Address, Note Counter. Вре-
мя,забираeмoe  проигрывателем - 5000 так-

Декомпилировать мелодию. Поиск ведется в 
файлах с любым расширением.
TO BEGIN +
Увеличение номера ноты, с которой размeс-
тится копируемый блок нот.
TO BEGIN -
Уменьшение номера ноты, с которой размeс-
тится копируемый блок нот.
SET UP
Установка конфигурации. Можно установить:

tracker),  печать  точек или нулей в окне
редактирования мелодии, цвета, beeper при
нажатии на клавиши, Kempston joystick.
