ZX Hard 00 - Измерения - очень полезная схемка для проверки печатных плат.
┌──────────────────────────────────────┐
│             Измерения.               │
└──────────────────────────────────────┘

(r) VTS'98
(c) инженер А.Белоусов, ЮТ'1/86.

Сейчас  я  нарисую  одну  очень полезную
схемку  для  проверки  печатных  плат из
журнала "ЮТ (юный техник) 1/86".
В  скобках будут даны номиналы для пита-
ния от +5V (в оригинале - +3V).

        ┌──┬───┬───┬───┬───┬─────────┐
      R1║R2║ R3║ R4║ R5║   │ VD2     │
        ║  ║   ║   ║   ║   └>├┐      │
        │  │   │   ├───┼──┐   │      о
        │   \│ │ │/    │  │ R7║   +3(5)v
        ├──</├─┴─┤\>───┤  │   ║
        │  э       э   │  │   │     gnd
        │    VT1,2   R6║  │ │/       о
  <─────┴─>├─┐         ║  └─┤\>┐     │
 щупы   VD1  │         │      э│     │
  <──────────┴────┬────┴───────┴─────┘
                 ─┴─        VT3

VT1...3 - КТ315, VD1 - Д9, VD2 - АЛ307
R1 - 300(510), R2,4 - 5.1k, R3 - 51k,
R5 - 3k(5.1k), R6 - 100, R7 - 100(300).


Принцип   действия:  если  сопротивление
между  щупами ( Rx ) менее 10 ом, то за-
горается светодиод. Это позволяет прове-
рить дорожки на обрыв либо замыкание.
Из-за низкого испытательного напряжения,
ограниченного  диодом  VD1 (около 0.3v),
все переходы п/п приборов на плате будут
заперты  ( R>>10ом ); также в маломощной
аппаратуре нету резисторов номиналом ни-
же 10ом (исключение - катушки/дроссели),
что  позволяет  использовать  пробник  в
'набитой' плате и при ремонте, НО ИССЛЕ-
ДУЕМЫЕ ЦЕПИ ДОЛЖНЫ БЫТЬ ОБЕСТОЧЕНЫ !!!

При  проверке  конденсаторов большой ем-
кости  пробник  даст единичный импульс и
затухнет (иначе - кондюк пробит).

3лементы  R1,Rx,R5,R6  образуют  измери-
тельный   мост,   в  диагональ  которого
включен компаратор на VT1,2. На VT3 соб-
ран буфер для индикатора (VD2).
