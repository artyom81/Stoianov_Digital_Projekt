Demo or Die 01 - Digital Design - Новый музыкальный редактор для цифровой музыки.
──────────────────────────────────────────
Привет.   На  протяжении  следующихсеми  килобайт  с Вами буду я, KENOTRON. Атемой   нашего  разговора  будет  редактор'DIGITAL DESIGNER'.         Вначале   некоторые  характеристикиредактора:
+
тип                     -
TRACKER
+
выходное устройство     -
COVOX
+
частота микширования    -
17 KHz
+
количество октав        -
3
+
количество каналов      -
3
+
количество инструментов -
16
+
количество паттернов    -
27
+
количество позиций      -
99       Редактор    поставляется   в   двухверсиях: для 128K - с чатотой микширования17KHz, и для 48K - с частотой 22(!) KHz.       Управление  стрелкой осуществляетсяс помощью KEMPSTON'a
или
SINCLAIR'a.       В  опциях  работы  с  диском курсоруправляется  клавишами CS+6, CS+7; 0,ENTER- выбор; CS+SS
- выход.       При   обращении   к   диску
BREAKпрерывает  этот  процесс, после чего можновозобновить его - RETRY
или выйти -
ABORT.     В верхней строке находятся опции:        1.
PROJECT,
в которой можно:
New
- очистить рабочую память для               создания нового модуля
Open
- открыть (загрузить) файл
Save
- записать на диск модуль
Clear
- очистить все паттерны. 2.
SETTINGS
- различные установки:       изменение порта, вкл/выкл SINCLAIR,       выбор скорости курсора, выбор длины       паттерна. 3.
ABOUT
- сообщит вам fast credits.       Ниже  экран  делится  на 2 части, влевой половине которого:
Play song
- проигрывает модуль с                самого начала
Cont song
- проигрывает модуль с                выбранной позиции
Play patt
- играет паттерн с начала
Cont patt
- ирает паттерн с позиции                курсора.
C
- удаляет выбранный инструмент
L
- дает возможность загрузить        инструмент.
Самый RULEZ        заключается в том, что можно        грузить 34-х килобайтовый (а может        и больше!) инструмент.
Выбор      текущего     инструментаосуществляется  стрелками  справа  от  егоимени. После загрузки текущего инструментаего номер увеличивается на единицу.    Loop
- установка признака                зацикливания инструмента
Begin
- смещение зацикливания от                начала (в HEX).
Edit patt
- выбор рабочего паттерна,                вкл/выкл    редактирования                осуществляется    нажатием                FIRE или SPACE
Octave
- выбор октавы
Step
- выбор, вкл/выкл шага                смещения по паттерну при                редактировании.
A,B,C
- включают/отключают каналы.       При  выборе  оставшихся трех опций,их  содержимое отображается в правой частиэкрана: 1.
POSITIONS                                                                                               Здесь      отображается      списокпозиций-паттернов,  который  скроллируетсястрелками, а также:
Lenght
- установка длины модуля в              позициях включительно
Patt
- выбор паттерна для установки              в списке
Loop to
- зацикливание  модуля с              выбранной позиции
Set
- запись выбранного паттерна в              список позиций
Insert
- вставка паттерна в список
Delete
- удаление паттерна из списка
Home,End
- в начало/конец списка
Autostep
- установка  автоскролла при               записи паттернов
Clear all
- очистить список позиций 2.
TRANSPOSITION
Song
- все   производимые   изменения            будут   относиться   ко  всему            музыкальному      произведению
Patt
- то   же,  но  к  выбранному            паттерну
Chann
- к  каналу, в котором находится            курсор
Range
- к отмеченному блоку.
All
- изменения   для  нот  с  любым            инструментом
Curr
- с текущим инструментом.
Octave up/down
- транспонирование                      на октаву вверх/вниз
Halfstep up/down
- транспонирование                     на полутон вверх/вниз
Change notes:
Source
- нота, которую надо                  изменить
Destination
- нота, на которую надо                  заменить
Change
- заменить первое вторым
Swap
- обменять их между собой
Change sample
- аналогично   предыдущему,                 но   все   относится   к                 инструментам,  а  также:
Del
- удаляет   все   ноты  с  source-          инструментом. 3.
EDIT
Range current channel
- отметка  блока                       будет производиться                       в  текущем   канале
Range current block
- отметка  во-всех                       каналах
Mark chan
- отметить канал,  в котором                стоит курсор
Mark patt
- отметить текущий паттерн.
Reset range
- снять отметку
Clear range
- очистить отмеченний блок
Copy
- скопировать  отмеченний блок в            буфер
Paste
- вставить  блок   из   буфера с            позиции курсора
Clear bufer
- очистить буфер
Name
- имя файла
Tittle
- можно писать что угодно
HOME
- установить  курсор  в  начало             паттерна.
Ну   вот,  с  опциями  разобрались.Теперь перейдем в нижнюю половину экрана -полю редактора.       Курсор    перемещаеся   с   помощьюкурсорных   клавиш
(CS+5..8),
а  такжеклавишами
I,L
-  вверх,вниз.  Возможнаустановка   курсора   с  помощью  стрелки,наведенной на нужную позицию с последующимнажатием
FIRE
.       В каждом канале содержится:
C-2 3A05          │  ││└┴───
параметры команды          │  ││          │  │└───── код команды          │  │          │  └────── номер инструмента          │          └───────── нота-октава       Выбор октавы производится клавишами1,2,3
.       Инструмент       можно      ставитьнепосредственно  в поле инструмента
(1-G)
.Если  его номер =
0
, то играется предыдущийинструмент заданной нотой.
Команд реализовано пока две:
A
- установка    скорости              проигрывания модуля (01-0F)
B
- признак конца паттерна (без              параметров).
В поле ноты:         клавиша
ENTER
- стирает ноту с                         инструментом         клавиша
R
- устанавливает                         команду глушения                         канала.       При нажатии
CS+9
происходит отметка(или снятие оной) в текущей позиции
канала(если  выбран  Range  current channel) илистроки
(Range current block).
Да, и еще две полезные функции:
SS+H
- устанавливает курсор в                    начало паттерна
SS+P
- проигрывает паттерн с                    позиции курсора.       Прервать     проигрывание     можнонажатием
FIRE, CS, SPACE, ENTER.       Вот в общем-то и все. Дерзайте!
