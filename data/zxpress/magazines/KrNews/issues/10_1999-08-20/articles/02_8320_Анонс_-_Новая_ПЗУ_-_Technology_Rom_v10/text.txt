  Пришла пора рассказать о происходящем
в нашей Лаборатории.

  Сейчас  ведутся  работы  над созданием
Technological  Rom  1.0  - прошивки ПЗУ,
которая  занимает место Basic'а 128. Она
призвана заменить устаревшую прошивку от
Profi Code Computers Club.

  Что будет в ней будет?

  Конечно, там будет нечто вроде

              Tr Dos 128
              Tr Dos  48
              Basic  128
              Basic   48
              Sts   6.2i
              Hard  test

  Никакой музыки в меню не будет (напря-
гает). Управление в меню:

  1) cursor
  2) sinclair1
  3) sinclair2
  4) Q and A
  5) S and X

  При нажатии cs+Reset произойдет преры-
вание программы (экран при этом не испо-
ртится), бордюр станет синим. Далее кла-
вишами  0-7  производится выбор страницы
для работы Sts. Будет выпущено несколько
сабверсий  для  Pentagon'а (256...1024),
Profi (256...1024) и Scorpion/Kay (256).
При  нажатии клавиш 8 и 9 Sts будет раз-
мещен в дополнительной памяти.
  При входе в Sts содержимое всех регис-
стров будет передано в монитор.
  При  прерывании  по  линии  NMI  будет
произведен  вход  в монитор с отслежива-
нием  адреса точки прерывания программы.
Для  этого  необходимо  убрать защелку с
порта #7ffd.
  В  прошивке  заложено несколько тестов
аппаратной  части:  памяти,  клавиатуры,
сопроцессора и др.
  Возможно  (если вместится) будет всто-
рен  какой-нибудь  простой boot -только
для  старта basic'а, вызываемый нажатием
ss+Reset.

  Присылайте  пожелания  на адрес редак-
ции.

P.S.:  Если  у кого есть прошивка TR Dos
5.12f для Pentagon'а 1мб -plz, пришлите
ее в Краснодар.                        █
