Я  тут  решил  немного  порассказать   о
сетях, т. к. ( не в обиду будет  сказано
), мало  кто  из   наших   Краснодарских
ZXNet'чиков   ясно   представляет   даже
структуру  подобных   сетей,  да   и   с
адресацией порядка нет.

В пределах города для мыла все  выглядит
примерно так:

            C-Hub <──> Шлюз(ы) <─>───...
      ┌────┘│   │            │к др.сетям
      │     │   │            │
    Hub0  Hub1  Hub2 ... etc │-ZXNet др.
    │ │                      │ городов
  ┌─┘ └─┐                    │
Node0 Node1 ... etc          │-Fido,iNET
 ││└───────────┐             │ етц...
 │└─────┐      │             │
point0 point1 point2 ... etc


Как    оно    между    городами     пока
рассматривать не будем.

Адрес внутри Краснодара записывается так

Kr/99.99
 │  │  └── номер поинта на узле8
 │  └───── номер узла в сети
 └──────── Krasnodar - есть предложение
писать Krn ( от Smart'а ), хотя по-моему
за    этим    сокращением    закрепилось
KrNews... В общем, как будет  нормальный
сисопник, тогда и решим ( планируется  в
районе Дня Программиста в начале июля ).

Теперь  о  международном  стандарте   на
адрес

500:8612/99.99
 │   └────────── код города
 └────────────── Россия

Теперь  о  том  как  оно  все   работать
должно.

Засылает Петя ( Kr/00.15 ) письмо Васе (
Kr/01. 09 ) на  свой  узел  (  00 )  под
именем  "Kr/0109"  с  расширением   "m".
Потом  СисОп  на  досуге   просматривает
почтовый  диск  и  видит  енто  мыло.  С
криками   "Непорядок!   "    во    время
форфардинга ( час почтового обмена между
узлами ) он засылает его  на  свой  Хаб.
Если такого еще нету ( в сети всего  3-4
станции ), приходится ему искать телефон
01'го узла и форфардить мыло  самолично.
Если же  есть  Хаб  (  Hub ), то  дальше
будет  разбираться  он. Если  01'й  узел
зарегистрировался  на  другом  Хабе,  то
надо будет засылать (  аналогично ) либо
сразу на этот Хаб ( 2-3 Хаба на  сеть ),
либо на самый главный центральный C-Hub.
Хаб  -  это  БиБиСя  для  СисОпов. Т. е.
приходит СисОп и  скачивает  мыло  сразу
для всех  своих  поинтов, которые  будут
потом приходить к нему и скачивать  свою
почту.
Посмотрите  на  картинку  вверху  -  так
понятней должно быть.

Для эх обязательно  должен  существовать
один центральный Хаб, где они линкуются,
либо  какой-то  узел   (   модератора ),
выполняющий эти функции.
