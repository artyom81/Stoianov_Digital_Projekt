Программа №11:
SPRITER
Программа
SPRITER
прeдназначe-
на   для  превращения  элементов
картинок  в  спрайты с цветовыми
атрибутами.   Среди   достоинств
этой программы можно назвать та-
кие,  как  возможность  создания
большого количества спрайтов (до
83)  с последующей записью их на
диск,   возможность  цикличeских
сдвигов    картинки,   просмотра
справочной  информации  о спрай-
тах,    автоматическая   вставка
спрайта  между  двумя  другими в
памяти.  Всё это позволяет надe-
ятся,  что
SPRITER
будет полезен
тем,  кому необходимо превращать
картинки в спрайты с атрибутами.
Всё управление программой осу-
щeствляeтся  с помощью "горячих"
клавиш.  Далее мы будем пeрeчис-
лять  их и описывать те функции,
которые по этим клавишам вызыва-
ются.
H - HELP
Вывод справки о "горячих" клави-
шах
SPRITER'
а  (на  английском
языке).
S - SAVE
Запись   созданных  спрайтов  на
диск в виде одного файла.
L - LOAD
Загрузка  картинки  для создания
на её базе спрайтов.
А - DOWN
Увеличение высоты спрайта, кото-
рый вы собираетесь создать.
Q - UP
Уменьшение высоты спрайта.
О - LEFT
Уменьшение ширины спрайта.
Р - RIGHT
Увеличение ширины спрайта.
U - PUT SPRITE
Вывести текущий спрайт на экран.
Т - UP SCROLL
Циклический скроллинг вверх кар-
тинки  (может быть полезен, если
спрайт,  необходимый  вам, нахо-
дится  не  в самом верху картин-
ки).
R - LEFT SCROLL
Циклический скроллинг влево кар-
тинки   (функция  похожа  на  UP
SCROLL).
F - LOOK SPRITE
Посмотреть параметры создаваeмо-
го спрайта.
Z - ERASE FILE
Удалить все спрайты из памяти.
N - NUMBER "-"
Уменьшение    номера    текущего
спрайта на 1.
М - NUMBER "+"
Увеличение    номера    текущего
спрайта на 1.
1, 2, З - SCREEN
Смена на экране рабочей трети.
G - GET SPRITE
Создать спрайт. Выдаётся сообще-
ние о том, что спрайт создан.
С - CLEAR SCREEN
Очистить текущую рабочую треть.
I - INFORMATION
Информация   о   созданных  вами
спрайтах,  количестве  свободной
памяти и т.д.
Е - EXIT
Выход из программы SPRItER.
