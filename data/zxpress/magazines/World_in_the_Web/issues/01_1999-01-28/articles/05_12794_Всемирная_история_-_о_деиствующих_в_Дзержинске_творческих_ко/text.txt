   А  сейчас, мы расскажем  вам о действующих в Дзержинске твор-
ческих коллективах. В настоящий момент известно 4 наиболее круп-
ных  группы:
Master Home Computers Group,  FaceOff Association,
Power Project, Galaxy Soft Inc.
√
Master Home Computers Group ( MHCG )
Дата образования:
июнь 1995
Состав:
- Moonwalker
( organizer, coder, musician )
- Raven
( coder )
- Warewolf
( artist )
- Frodo
( coder )
- Time Keeper
( coder, artist, musician, hardware )
- Black Angel
( coder )
- Bear
( musician, hardware )
Что выпустили:
- Visual Player v0.5
        - Smak Megademo 
        - IZX
( Hobeta )
- Fast Breed
( logic game )
- Katharsis Eurodemo
( представлено на FUN_TOP'98 )
- Worm
( game )
- HardCore 4x4
( chank gfx editor )
- many intros
Проекты:
- X-Files
( quest game )
- Мураши
( военно-насекомая игра )
- Virtual Worlds
( disk magazine )
Рецензия: 
        Довольно сильный код, отличная графика и музыка.
        Программы,  выпущенные  MHCG  оставляют приятное  
        впечатление.
√
FaceOff Association ( #FOFF )
Дата образования:
Осень 1998 года.
Состав:
- Max Klm
( lamer, artist, gfx converter, organizer )
- Mr.First
( coder, musician, hacker )
- Mr.Second
( artist )
- Woland
( coder )
- Organism
( coder, artist )
- SLG
( coder, hacker, zx-net coordinator,  
                     hardware )
Что выпустили:
- FunTop98 Music collection 
        - FunTop98 4k intro collection
        - Clinic Death Eurodemo ;-) 
        - BMP->ZX converter
Проекты:
- NetWalk
( logic game )
- SWAT
( strategic game )
- WinBoot
Рецензия: 
        T. к. группа недавно образовалась, рецензию всей
        группе  дать невозможно, я  дам рецензию каждому
        члену:
Max Klm
- портит чужие программы своими вставками через 
            disc-doctor, немного рисует и поет. ;-)
Mr.First
- неплохо кодит, очень плодовитый музыкант, ломает
            все демы, которые попадаются ему на глаза.
Mr.Second
- лентяй-художник, который, впрочем, неплохо рисует.
Woland
- когда-то кодил, но совсем обленился, пишет только 
            небольшие процедурки, маньяк дисковых лоадеров.
Organism
- кодит, довольно неплохо, но с ним тяжело работать.
SLG
- Постоянно пишет всякие маньяцкие скроллы под немыс- 
            лимыми углами, ломает всякую бяку, курит, пьет и
            матом ругается.
