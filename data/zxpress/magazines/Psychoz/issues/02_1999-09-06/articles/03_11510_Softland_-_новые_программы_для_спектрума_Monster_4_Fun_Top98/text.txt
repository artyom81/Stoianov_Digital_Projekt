    Вот  и  вышла  финальная  версия  SED  Editor'а.  Юзайте  на
здоровье!  С  помощью  SED'а  (Sprite EDitor) вы сможете творить
чудеса  ;).  В  нем  существует множество функций, которые вы не
встретите  ни  в  одном  из  подобных  редакторов.  Например: 3D
вертелка,  автовыдирание  спрайтов  из картинки и многое другое.
Кстати,  в  программе  есть  два  (2)  встроенных help'а. Первый
вызывается  клавишей  <H>  при  редактировании спрайта, а второй
можно  поглядеть  из  меню  с  иконками,  тоже  нажав кл. <H>. В
редакторе  существует  лишь  один  недостаток  ->  можно  коцать
спрайты,  размером  не  более  2x2.  Это  потому, что изначально
редактор писался для игрушки.
.Real Commander v1.8.
status ...... system
real size ... 64 sec. 
trademark ... Real software'999
Idea, Code, Design ....................................... Pawel
Отличия от версии 1.7
√ Поддержка всех 4-х частотных таблиц Pro Tracker 3.4+
√ Полноценный частотный анализатор.
√ Копирование  файлов  как  в порядке каталога, так и в порядке
маркировки. Для маркировки файлов больше не используется 7-й бит
первой буквы файла.
√ Выбор маски в SetUp'е.
√ Поддержка аппаратных CMOS часов.
√ При  копировании  группы  файлов  с  одинаковыми  именами,  на
принимающем диске они не затирают друг друга.
√ Поддержка резидентного  RamDisk'а для машин  с  памятью  свыше 
128к.
√ При проигрывании музыки, вернуть табличку с названием музыки и
частотными анализаторами можно клавишами CS+P.
√ Исправлена работа с файлами нулевой длины на RamDisk'е.
√ Переход к просмотру/прослушиванию  следующего файла по клавише
Enter.
√ При   копировании  на  1 - ом  дисководе  можно  не  проверять
принимающий диск.
√ Просмотр экранных файлов без атрибутов.
В  комплект  поставки  RC  входит  rom  предназначенный  для
прошивки  вместо  128 бейсика, эмуляторщикам все намного проще -
достаточно  скопировать  этот  файл  1:1  в MS-DOS и прописать в
конфигурации  эмулятора.  Перед  прошивкой  необходимо настроить
конфигурацию  RC  под свои нужды. Два байта SETUP'а находятся по
смещению  #3B,  в  самом командере они находятся по смещению #5.
Вам необходимо настроить в SETUP'e командер и запомнив два байта
конфигурации вписать их в файл прошивки.
