Inferno 04 - Inferno - Об оболочке.
Итак немного об этом монстре что вы сей-
час вероятно :) листаете.

  Как   вы  видите  эта  оболочка  немного
отличается от того что было раньше...
  По  сравнениюю с третьим номером, теперь
вы  можете просмотреть название композиции
и имя автора только в том случае, если во-
йдете в Set Up (Установки).

  Управляется вся эта байда от мыши, джой-
стиков и клавиатуры, о  чем и сообщается в
Helpе, а  вот  о  листалке  стоит  сказать
особо.

  Курсорными клавишами можно двигать текст
как  построчно, так  и  постранично, кроме
того, для  любителей погонять мышь по ков-
рику, экран разбит на контактные зоны:

Выход     |                    | хранитель
          |                    |    экрана
          |                    |
          |                    |
          |     построчно      |
          |       назад        |
          |                    |
          |                    |
          |                    |
          |                    |
          |                    |
страница  |____________________|  страница
назад     |                    |    вперед
          |                    |
          |                    |
          |                    |
          |                    |
          |     построчно      |
          |       вперед       |
          |                    |
          |                    |
          |                    |
          |                    |   Быстрый
Установки |                    |      тест

Как  и в главном меню при просмотре текста
по  углам  вызывается  всякая  байда вроде
хранителя экрана и быстрого теста, кстати,
тест  действительно быстрый... быстрее не-
куда :)

 Также  в  главное  меню можно выйти нажав
клавишу "E"  или  кнопку "откат"  мыши или
один из шифтов.

 По  поводу грызуна: первая нажатая кнопка
идентифицируется   как "выбор",  если   вы
вдруг  облажались и нажали не ту кнопку то
пункт  меню "Развернуть кнопки" быстро всё
исправит.
