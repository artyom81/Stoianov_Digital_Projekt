Типы файлов, определённые в ZXVGS
Происхождения расширений:
new
- новое, определённое для нужд ZXVGS
old
- совместимость сверху вниз
other
- с других платформ.
Возможные операции:
load
- файл может быть запущен с помощью
RSX,
data
- данные из файла могут быть загруже-
ны с помощью RSX,
run
- ZXVGS может запустить такой файл,
screen
- файл может использоваться для
хранения экрана.

Структура файлов "*.BZX", "*.CZX", "*.DZX"
одинакова.
То же можно сказать о файлах "*.GZX",
"*.HC", "*.HP".

Двухбайтные числа в заголовках (start,len,
prog) лежат в порядке L, H. Это не относи-
тся к сжатым файлам.
