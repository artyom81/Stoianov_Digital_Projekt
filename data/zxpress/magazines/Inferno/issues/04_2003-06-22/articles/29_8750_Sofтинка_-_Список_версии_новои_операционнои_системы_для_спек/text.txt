Inferno 04 - Sofтинка - Список версий новой операционной системы для спектрума ZXVGS.
Версии ZXVGS
MB2 - MB-02
Использует  банки  памяти с 5 по 7. 1804kB
диск имеет структуру CP/M. Изменений аппа-
ратуры не требуется.
PEN - Pentagon 512kB
Предназначено  в основном для тестирования
на эмуляторах: X128,R80 и ZX_EMUL. Поддер-
живает  Kempston Mouse. Использует диски в
физическом формате TR-DOS 640kB,но с логи-
ческой структурой CP/M для Scorpion.

Распакуйте архив "ZX???PEN" в директорию с
эмулятором, после  чего  запустите  скрипт
"ZXVGS.BAT". Эмулятор  распознаётся скрип-
том.
PL3 - ZX Spectrum +3
Использует  дисководы  ZX+3 (также 3.5") и
диски в форматах CPM22QED. Также может уп-
равлять  жёстким  диском (например,  через
плату  YAMOD.ATBUS или 8255). Оригинальная
система PLUS3DOS уничтожается.



Для запуска на реальной аппаратуре необхо-
димо произвести некоторые внутренние изме-
нения.Оригинальные ПЗУ должны быть замене-
ны на EPROM и SRAM, каждая по 32kB. Поско-
льку ULA выбирает внутренние ПЗУ только на
чтение,нужно провести дополнительный тракт
для правильной работы SRAM. Возможна также
инсталляция ZXVGS на ZX Spectrum +3 посре-
дством внешнего интерфейса.Для использова-
ния  дисковода  3.5"  необходимо поместить
импульсный диод между сигналами /MOTOR (K)
и /READY (A) (большинство  3.5" дисководов
не выдают сигнал /READY).

Для запуска этой версии на эмуляторе,банки
ROM2  и  ROM3  должны быть доступны на за-
пись. На  сегодняшний  момент это возможно
только на эмуляторе XZX 2.9.0,который тре-
бует некоторых модификаций (изменённые ис-
ходные файлы включены в архив).Используйте
QTRANS.EXE  для  создания образа диска для
эмулятора - укажите  в  качестве параметра
имя  каталога с файлами (это будет также и
именем файла ".DSK").Вы можете поместить в
каталог любой файл, который желаете скопи-
ровать на образ диска CPM22QED. Для реаль-
ного железа используйте CPDWRITE.EXE,кото-
рый  форматирует  диск и копирует образ на
реальный диск.
PLD - Plus D
Использовет диски в форматах CPM22QED.
SAM - Sam Coupe'
Использует систему CPM22QED для управления
дисками. ZXVGS может быть запущен из кома-
ндной  строки через "ZXVGSSAM". В качестве
параметра может быть указан файл, запуска-
емый после ZXVGS. Обслуживает оригинальную
Sam Mouse.
SCO - ZS Scorpion (ROM)
Использует  расширенную  память (512kB)  и
турбо-режим процессора.
TMX - Timex Interface (TI-of-TTL)
Версия предназначена для интерфейса, кото-
рый  работает  наподобие  Timex  Interface
M-397, обычно  используется для соединения
Timex  FDD 3000 (сокращённо TF). Новый ин-
терфейс имеет расширенную память и исполь-
зуется для подключения второго компьютера,
работающего как file server. Второй компь-
ютер  может  быть не только TF - с помощью
специального  кабеля  Amiga  или  PC могут
быть  подключены  через параллельный порт.
ZXVGS может также использовать мышь, подк-
люченную к серверному компьютеру.

Для запуска серверного ПО:
*
TF: запустите CPM22QED,после чего вве-
дите в командной строке "ZXVGSTF",
*
Amiga: запустите ZXVGS.server,
*
PC: запустите  ZXVGSTSR.COM,  а  потом
ZXVGSLPT.COM (обычно они оба вызываются из
скрипта).
UPB - Unipolbrit Komputer 2086
(Timex Computer 2068)
Эта версия может работать с внешним серве-
ром (см. версию TMX) или с массивной памя-
тью, подключенной непосредственно (включая
FLASH EPROM картридж).
WRJ - Warajevo 2.51 emulator
Это  версия поддержки для реальных версий.
Обеспечивает  доступ  к  файловой  системе
хоста из уровня эмуляции.

Распакуйте  архив  "ZX???WRJ", после  чего
скопируйте подкаталог "ZXVGS" в корень ди-
ска C:. Для запуска ZXVGS вызывайте скрипт
"ZXVGS.BAT".
