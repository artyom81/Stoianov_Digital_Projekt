I N S T R U C T I O N S 
LINEZIP IS A UTILITY TO HELP YOU SAVE MEMORY BY "ZIPPING" TOGETHER LINES OF BASIC. ONLY A FEW BYTES ARE SAVED WITH EACH "ZIP" BUT IN LARGE PROGRAMS  WHERE THE SAVING OF MEMORY IS IMPORTANT,SAY IN AN ADVENTURE OR A  DATABASE,  THESE BYTES  BUILD UP TO A SIGNIFICANT SAVING. BASIC ALSO RUNS FASTER IF FEWER LINE NUMBERS ARE USED.
IT IS RECOMMENDED THAT YOU  SAVE THIS MASTER PROGRAM AS A BACK UP AND GENERATE AND  SAVE A  WORKING COPY BY USING THE OPTIONS AT THE MAIN  MENU.  THE  WORKING  COPY, SAVED AS "ZIP", CONSISTS OF ONLY A SINGLE LINE AT  LINE 7777 WHICH CAN FOR BE EASILY MERGED  IN  TOYOUR OWN PROGRAMS FOR USE.


TO USE LINEZIP MOVE LINE 7777 TO A POSITION JUST  IN FRONT OF  THE LINES YOU WISH TO 'ZIP' TOGETHER BY EDITING  IT AND RE-NUMBERING TO A NUMBER JUST BELOW THE FIRSTLINE YOU WISH TO 'ZIP'.ENTER RUN AND THE LINE  NUMBER,  ERASE  THE ZIP LINE (BY JUST TYPING  IN  ITS NUMBER)AND FOR REMOVE THE 'ZIP' IN THE MERGED LINES (A SERIES OF COLONS!) BY EDITING THEM OUT.
HERE IS AN EXAMPLE:-
SAY YOU HAVE THE  FOLLOWING  TWO LINES  THAT  YOU WISH TO  MERGE TOGETHER:-

3200 PRINT 'TESTING'
3250 PRINT 'ZIPLINE'


1. EDIT OUT  LINE 7777 AND  ALTER   THE LINE NUMBER TO SAY,  3150 
2. ENTER  RUN 3150 OR GO TO 3150 
3. ENTER  3150 TO  ERASE  LINE 3150 
4. ONLY   LINE 3200  WILL  REMAIN    AND SHOULD LOOK LIKE THIS:-

3200 PRINT "TESTING"::::: PRINT "LINEZIP" 

5. ERASE 4 OF THE COLONS  IN  THE   MIDDLE OF THE LINE  TO  FINISH   THE OPERATION. IN THIS EXAMPLE   YOU COULD ALSO ERASE THE  STH   COLON AND REPLACE THE  SECOND    PRINT STATEMENT WITH A "'" TO   LOOK THIS:-
3200 PRINT "TESTING"'"LINEZIP"


OBVIOUSLY,  IN THIS  EXAMPLE  YOU COULD JUST AS EASILY RE-TYPE THE LINE ANYWAY BUT WITH MUCH LARGER LINES IT IS FAR EASIER (AND LESS PRONE TO TYPING  ERRORS) TO  USE LINEZIP. 
NOTE THAT  IF YOU  CREATE A  LINE THAT IS  BIGGER THAN THE SCREEN, YOU WILL  NOT BE ABLE TO SEE  THE CURSOR IF YOU TRY TO EDIT  BELOW THE BOTTOM VISIBLE LINE SO  WHEN ZIPPING TOGETHER SEVERAL  LINES, ZIP THE LAST TWO TOGETHER FIRST,EDIT OUT THE COLONS AND  FOR  GO BACK TO THE PREVIOUS LINES.
POKE 23608,0 IS VERY  HELPFUL TO GET RID OF THE WARNING BUZZ THAT SLOWS THINGS DOWN  CONSIDERABLY!
