 Приветствую всех читателей новой модемной газеты LPRINT.Я хочу
рассказать о доработке C-DOS модема, именуемой в народе Релюшка.
Итак, на плате модема есть резистор 1 МОм,который создает нагру-
зку в линии для того,что бы сервер реагировал на звонок.Так вот,
если его уменьшать, то сервер будет лучше реагировать на  вызов
абонента /у меня стоит 500 кОМ/, но при этом ухудшится качество
связи. Для того, что бы этого не произошло, нужно в то время ко-
гда "трубка" на модеме поднята - отключать этот резистор.Этим и
занимается Релюшка.Рисунок показывает фрагмент платы,там где на-
ходится разъем для телефонной линии:

       \/                      o────────┐
     ┌┐┌┐┌┐ │                ┌┐  ┌┐ │  ┌┴┐ Выпаяный резистор,
     ││││││ │                ││  ││ │  │ │ который нужно
     └┘└┘└┘ │                └┘  └┘ │  └┬┘ отключать
            │             ┌────o    │   │
     ┌────┐ │             │  ┌────┐ │   │
     └────┘ │    Релюшка  │  └────┘ │   │
     ┌────┐ │    на плате │  ┌────┐ │   │
     └────┘ │        o o  │  └────┘ │   │
      ┌┐ ┌┐ │       o│o o │   ┌┐ ┌┐ │   │
  ────┴┴─┴┴─┘      ──┼┼───┼───┴┴─┴┴─┘   │
                     ││  ┌┼──────┐      │
 Вид со стороны      │└──┼┼─o o  │      │
 платы               │   │└o o o─┼──────┘
 /так было/          │   └───┼───┘ Вторая Релюшка
                     └───────┘    вид со стороны выводов

Если Релюшки откажутся срабатывать вместе, то тогда их нужно бу-
дет соеденить последовательно и запитать от 12 вольт. Так,напри-
мер, сделано у Lam'ы:

   ┌ Разрезать дорожку от +5В /длинная такая/
   │       
   │      │  ┌────┐ │  Последовательное соединение Релюшек
   │ o o  │  └────┘ │
  \/o o o │         │ <---
  ─x──┤   │   ┌┐ ┌┐ │  Между реле и трансформатором
  ────┼───┼───┴┴─┴┴─┘  есть диод, который нужно выпаять
      │  ┌┼──────┐
      └──┼┼─o o  │
         │└o o o─┼──На резак
         └───┼───┘
 +12В ───────┘


 Теперь теория: вторая Релюшка работает на размыкание цепи, т.е.
в исходном состоянии /когда "трубка" положена/ ее контакты в це-
пи резистора замкнуты и таким образом он включен. При  поднятии
трубки - она срабатывает вместе с Релюшкой на плате и разрывает
цепь резистора, который теперь не портит связи.
 Релюшка устанавливается такая же, что и на плате: РЭС 49 с нап-
ряжением питания 6В.
