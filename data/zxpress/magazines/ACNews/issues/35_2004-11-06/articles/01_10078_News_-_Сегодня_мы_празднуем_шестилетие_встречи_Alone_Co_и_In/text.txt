───-+ of alone coder +──────────────────────────────────────────

Ну  как вам новый редактор? Не находите, что великоват? :) Это в
нём  уместились все функции, которые были за его историю, кроме,
разве  что, экрана  384x304. Версию  с  таким экраном я для себя
соберу одним движением пальца :) У вас такая возможность будет в
следующий  раз. Да, SMT  велел  для просмотра 384x304 включать в
Unreal Speccy широкий бордюр.

Сегодня мы празднуем шестилетие встречи Alone Co. и Invaders Gr.
Оно  может  считаться  праздником основания расширенного состава
Invaders Gr., тогда  ещё  не i8. Мы и тогда готовились к выпуску
Guide, и  тогда  тоже  материала  не хватало :) Вообще говоря, я
привык (не знаю, как John и Basil ) делить все события на Speccy
как  на  происшедшие  ДО осени 1998 (до ZX-Guide ) и происшедшие
ПОСЛЕ. Старичков, успевших выпустить что-то ДО, осталось мало...
Но они есть :)

Так вот, первая версия AC Edit (v0.03) была выпущена ДО :) Тогда
он  ещё  назывался  просто  EDIT  и выходил в исходниках, но без
автосборщика, в  отличие, например, он  v0.87. Чего  же  я успел
добиться  за прошедшее время в плане развития редактора? (Погля-
дите  на  номер  версии  - судя по нему, до полной остался ровно
год!)  Собственно, чего-то  добился, но  трудно  перечислить все
положительные  стороны. Возьму на себя труд перечислить различия
по сравнению с v0.59. Которая "AC Edit'у три года", помните?

convert/plug-in:
- после выполнения плагина не контролировалась (оставалась
прежней) длина области спрайтов.
+ встроенные конверторы X=export to, O=import from:
 конверсия из/в ISO;
 экспорт в KOI, WIN;
 импорт из BUG (toWINunKOI), из транслита, из UNICODE (UTF-8),
из DOC и из кодировки, где символы имеют вид "=XX";
 конверсия цветов в управляющие коды принтера.
+ функция разламывания длинных строк по пробелам до правой
границы и удаления нулей в тексте.
- до печати прерывания выключаются, после включаются.
+ плагин 2kolonki - разбиение текста на 2 колонки.
+ суперигра IMPETRIS!
+ особый плагин печати acePRN by MaXx/XTDM.

design/screen:
+ 64-символьный шрифт доработан, чтобы в фильтре эмулятора все
буквы были разные. Шрифт также значительно более соответствует
стандартному для ascii!
+ поддержка 384x304!
* в режиме 512x192 спрайты отключены.
+ переключение hex/dec режима для 5-значных чисел.
* фоновый цвет теперь определяет и фон для цветных букв.
+ строка статуса!
+ утолщенный шрифт 8x8.
+ X-координата и длина строки трёхзначные.
- убран глюк склеивания строк с табуляциями.
- неправильно ставился символ переполнения строки, если в строке
есть табуляции.

compile/init:
+ концепция "сделай сам": из исходников каждый может собрать
себе облегчённую версию ACE со своими настройками!
+ во встроенном хелпе видно только прикомпилированные функции.
+ в инициализации прописывается REPDEL=15 для скорострельности
клавиатуры.
+ изначальная загрузка музыки и шрифта (по вашему выбору, <f>
или <C>)
+ загрузка с CS - пропустить поиск шрифта и музыки.

keyboard:
* Home=SS/Q, End=SS/E, по стандарту.
+ &=ё во всех режимах клавиатуры (Macros)
- исправлен дребезг стрелок в режиме "русские на цифрах".
+ раскладка для эмуляторов и xt клавиатуры. Символов в русском
режиме доступно больше, чем в пцшных редакторах!
- при курсоре левее левой границы Enter съедал часть строки.
+ кнопки "слово вправо", "слово влево". Стараются двигать курсор
по пробелам (для нужд раскраски)!
+ Undo работает в обе стороны.
+ Undo восстанавливает строку, удалённую по SS/Graph.
* убрано ограничение на растягивание строк с окончанием на знак
препинания.
- правильно растягиваются цветные строчки.
+ в режиме вставки Break раздвигает цветовые коды!
+ появилась функция "курсор вниз" для раскраски абзацев или
ввода лёжа!
+ украинский режим клавиатуры!
+ GotoEnd идёт на последний символ.
+ звук в лат. и рус. режимах различается!
+ теперь через бейсик переопределяются почти все кнопки.

format/setup:
+ выход из сетапа также по SS/Enter.
+ цвета с bright'ом, коды 14-20 (SS+цифры в граф.режиме).
+ анимированные спрайты! Ускорен вывод спрайтов.
+ строчки >255 байт можно ломать, не сходя с них (символы в
хвост добавляются автоматически)!
+ грузит LF и LFCR тексты в любом режиме!
+ опция оптимизации пробелов табуляциями!

disk:
+ Break=Load (раздвижение символов тоже работает)!
+ горячее сохранение клавишами CS/Enter!
- изменён лоадер: сообщений TR-DOS больше не будет, будет повтор
текущей операции, Break=отмена.
+ выбор дисководов через #3d13 #18!
+ в Save можно выбрать дисковод цифрами 1-4, если переместить
курсор в конец имени (раньше там был глюк)!
+ полная поддержка трёхсимвольного расширения!
* при загрузке текста сначала пробуется режим 42 символа в
строке - незаменимо при отключенном сетапе.
+ просмотр и вставка экранов в текст!
+ в дисковом меню видно имя файла в памяти.
+ видно число секторов в файлах.
+ отображается текущий дисковод.
+ вывод каталога по столбцам.
+ разноцветные файлы!
+ фильтр файлов в каталоге убран - можно грузить и <H>, и <B>, и
удалённые файлы.
- после Merge не перекодировались окончания строк.
* Merge ускорено.
+ запуск бейсик-файлов!
+ загрузка шрифта 4x8 "f".
+ Delete file
+ Rename file. Восстанавливает стёртые файлы!
+ Move disk!
+ при переходе на вторую страницу каталога диск не
перечитывается.
+ Load спрашивает подтверждения на сохранение текста.
+ проверка переполнения диска при записи - индицируется красным
бордером.
- при очистке текста (1/3/5) сбрасывается флаг изменённости
текста и меняется его название (инкрементируется последний
байт).
+ встроенный плейер PT3.6x.
+ Hrust packer!
+ можно грузить музыку, упакованную Hrust21.
+ музыка при распаковке текстов не выключается.

find:
+ длина строки поиска 16 символов вместо 9.
+ в режиме Find можно вводить коды с помощью кнопки Ext.
+ SS/I - продолжение поиска.
+ подсчёт количества замен при поиске!
+ при поиске (не замене) пробел значит любое число
пробелов/цвета/CR/LF!
+ если при поиске включен Caps Lock, то поиск ведётся без учёта
регистра!
+ в окне Find/Replace по умолчанию стоит комбинация для замены
русской Н на латинскую.
+ поиск/замена В БЛОКЕ, если курсор находится в этом блоке!
- replace зацикливался на замене на строку, содержащую в
подстроке искомую.

Вообще, ACE  самый живучий проект на Speccy. Он ни разу не прек-
ращался  больше, чем на 2 месяца. Когда не пишется сам редактор,
расширяется  todo list,  благодаря  которому, например, за  один
вчерашний  день  сделано аж 17 изменений. Правда, не стоит гово-
рить, что  ради  этого  вчерашний день пришлось растянуть с 4:20
утра до полуночи :)

Стараниями KSA-7G к т.н. пентагону Диссонатора подключен сначала
CD-ROM, а  затем  HDD. HDD заработал только один из двух, и то -
только  при подключенном на тот же шлейф CD-ROM'е. Причём CD-ROM
должен  быть  Master, иначе  программа  CD-Walk не работает. Для
HDD, за  исключением  iS-DOS, который  я не хочу ставить, у меня
пока  имеются  только  программы Storer by SAVC (очистка) и Hard
Disk Search&Test 0.05 by Breeze (определение параметров).

Dissonator  обещал  прийти на этой неделе посмотреть, но не при-
шёл. Врун. Работает не говорит где с утра до вечера.
