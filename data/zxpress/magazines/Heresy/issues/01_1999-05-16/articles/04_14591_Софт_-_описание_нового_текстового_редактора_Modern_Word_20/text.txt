  + основная фишка этой версии - конвертор стандартного
    спектрумовского экрана в ascii, написанный SairOOs'ом,
    (thanx!). Конвертация происходит по таблице градаций
    подзагружаемых из файла "wordascii" (грузятся только первые
    16 байт).
  + подзагружаемый блок "wordfont"  с фонтом в формате одной
    трети спектрумовского экрана и, соответственно размером 2048
    байт. При отсутствии этого файла будет использован
    стандартный фонт. Использование стандартного фонта наиболее
    желательно!
  + функция "bye..." возвращается по стандарту Real Commander'a:
        out (#7ffd),#17  ; включение 7-ого банка
        cp #ffЗ1,(#c000) ; проверка на ld sp,#??ff в #c000
        jpz #c000        ; если да, то стартуем RC
        out (#7ffd),#10  ; иначе включение нулевого банка и
        push #0000       ; рестарт компьютера с адреса #0000
        jp #3d2f         ; в TR-Dos'е!
    Проще говоря: если вы запустили "ModernWorD" из Real 
    Commandera, то в него и вернетесь!
  + теперь листать опции и файлы в менюшках можно не только
    "вверх" и "вниз", но и убыстренно: "вправо", "влево".
  + загрузку теперь сопровождает cool'ьная картинка.;), которая
    называется "Е-мое, что ж я сделала-то???"
  > при покушении на нулевой трек был убит особо опасный баг,
    замеченный в предыдущих версиях.
  - в связи с новым статусом программы введен регистрационный
    ключ. При его отсутствии программа работает нестабильно,
    но тексты вроде не портит. О способах получения 
    вышеупомянутого ключа смотри приложение 2.

+++ народ, ХАЛЯВА !!! Поигравшись с регистрационым ключом и
    одарив всех своих друзей копиями, решил эту версию выпустить
    уже без ключа. Так что юзайте на здоровье!
  + внизу появилась дополнительная информационная строка. в ней
    содержатся следующие полезные сведения:
      - код текущего символа (перенесено с верхней панели).
      - размер маркированного блока текста.
      - размер всего текста.
    соответственно размер видимого текста уменьшен на одну
    строку и теперь равен 22 строкам.
  + опция
(сохранение параметров настроек): на
    диске создается/перезаписывается файл word_ini который
    синициализирует ModernWorD при следующем запуске. Кроме
    параметров самого меню setup'a отгружаются параметры
    формата, регистры rus/lat, ins/ovr, caps и текущий drive.
  + функция авто-скобки: при нажатии символа "(" печатается ()
    и курсор помещается внутри скобок. переключается опцией
    "autoparenthese" в setup'е.
  + при загрузке текста происходит проверка на соответствие его
    стандартному формату, при обнаружении несоответствий
    предлагается произвести коррекцию. переключается опцией
    "text checking" в setup'е.
  + функция проверки качества записи текста на диск (для
    надежности отключается также процедура fast_stop WG93).
    переключается опцией "quality save" в setup'e.
    Предпочтительнее держать эту опцию включенной дабы не
    пропали втуне ваши усилия ;)
 c+ через коммандную строку клавишей "A" маркируется весь текст.
  + при перемещении курсора влево при достижении начало строки
    курсор перекидывается вверх на одну строку на последний
    ее символ.
  + через setup можно переключить режим подтверждения удаления
    блока текста по Ext+D (special option 4 Unbeliever, который
    постоянно попадает по клавише D вместо F ;).
  + так как при выделенном блоке текста опция Save file 
    скидывает на диск только выделенную часть, то теперь в
    дисковом меню, в таком случае, опция Save file заменяется
    на Save BLOK. Будьте внимательны !
  + небольшие изменения и доработки...
  > если при форматировании изменялись параметры формата, то на
    крупных текстах modern мог тотально заглючить - пофиксено.
  > при переключении кодировок в ЯВЕРТЫ пропадала возможность
    набрать заглавные Ч,Э и некоторые другие символы - пофиксено
  > сделана небольшая доработка промотки текста вверх и вниз:
    теперь верно отображаются текущие номер строки и код символа
  > после удаления строки по "L" некорректно выставлялись
    некоторые переменные - исправленно.

 Приложение 4 |  аКа  обзор клавиш управления
---───────────┘
    Описание основных функций работы с текстом вызываются из
коммандной строки клавишей "H"(elp).
Ext.mode     - Вызов коммнадной строки.
True Video   - Вверх на страницу.
Inv. Video   - Вниз на страницу.
Ext + Edit   - Режим графических элементов.
Ext + Del    - Удаление строки от курсора до левого края экрана.
Graph        - Удаление символа справа от курсора.
Sym + Q      - На левый край экрана.
Sym + E      - В конец строки.
Sym + W      - Переключение режима "insert"
    При отмеченном блоке текста операции (format, save, etc)
производятся только с этим блоком. Будьте внимательны!
