..:: 3. Турбо 2+ версии 7.10 ::..
Параметры компьютера:
- CPU: Z80A(B,H) (3.5/7.0МГц).
- ROM: 64/128Кб (расширяется до 1024Кб).
- RAM: 1024Кб (4 окна проецирования. Это справедливо и для ROM).
- VIDEO: ZX(256x192); Hardw.Multicolor (640x200); EGA (320x200);
  Text (80x25).
- Палитра: 16 цветов одновременно из 64-х (во всех режимах).
- Звук: AY-3-8910(12)/YM-2149, DAC(covox), ADC, Beeper.
- Интерфейсы: FDD, IDE, RS-232, LPT (порт #FB), External, TAPE,
  XT/AT-keyboard.
- Клавиатура: AT-keyboard интегрированный контроллер, поддержка
  стандартной клавиатуры, Sinclair-Interface 2 джойстики.
- IDE: Интегрированный контроллер (поддержка жестких дисков и
  CDROM приводов).
- OS: Basic48/128; TR-DOS 5.03; CP/M 2.2; iS-DOS; TASiS; vTRDOS;
  поддержка резидента.
- vTRDOS: запуск/сохранение программ, адаптированных для
  BetaDisk FDD, с жесткого диска и CD.

Параметры платы:
- Размер: 335x190 мм - можно установить в корпус AT-Minitower.
- Вид: зеленая (т.е. С ЗАЩИТНЫМ ПОКРЫТИЕМ).

Неофициальный Интернет сайт, посвященный АТМ:
http://atmturbo.nedopc.com

..:: 4. Музыкальное расширение TurboSound ::..
Устройство предназначено для расширения музыкальных возможностей
Спектрум-совместимого компьютера при сохранении совместимости со
всем предыдущим ПО. Вместо одного музыкального процессора
используется два, т.е. шестиканальная система синтеза звука.
Плата подсоединяется с помощью разъема на посадочное место
музыкального сопроцессора на плате и не требует использования
паяльника для установки.

Для создания музыки с использованием Turbo Sound можно
использовать ProTracker 3.7x
(см. на http://trd.speccy.cz/system.htm).

..:: 5. Музыкальное расширение TurboSound FM ::..
Устройство предназначено для расширения музыкальных возможностей
Спектрум-совместимого компьютера при сохранении совместимости со
всем предыдущим ПО. 
TurboSound FM позволяет использовать шесть звуковых каналов 
вместо стандартных трех каналов у AY8910/12 или YM2149. И, 
дополнительно, шесть программных каналов с FM генерацией звука. 
Устройство устанавливается через соответствующий переходник на 
посадочное место AY8910/12..
Для создания музыки с использованием Turbo Sound FM можно 
использовать TFM Music Maker 
(см. новости на http://www.nedopc.com/soft.php).
